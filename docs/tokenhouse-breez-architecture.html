TokenHouse Breez Architecture

Vite + React chat client streaming member chat completions via TokenHouse (OpenAI-compatible SSE)

Client (this repo)

  • Vite + React SPA; config from VITE_TOKENHOUSE_* or settings UI
  • Persists URL, member API key, and model in localStorage
  • Streams assistant deltas by parsing data: SSE lines from the response body
  • Sends session_id per tab for conversation continuity

TokenHouse API surface

  • POST {baseUrl}/api/v1/chat/completions
  • JSON: model, messages, stream: true, max_tokens, session_id
  • OpenAI-style streaming chunks in choices[0].delta.content
  • Terminates stream on data: [DONE]

Security notes

  • Member API key sent as Authorization: Bearer … on every completion request
  • Keys in browser storage are visible to the user profile; use dev keys or short-lived tokens appropriately
  • AbortController cancels the fetch + reader when the user stops generation
